# GrantAI Pro

> 🎬 애니메이션·영상 국가 지원사업 지원서 자동 작성 플랫폼 (프로토타입)

KOCCA, KOFIC, KOMACON, 문체부, 경기콘텐츠, SBA, 부산영상 등 7개 기관 통합

## 실행

```bash
npm install
npm run dev
```

## 배포

Vercel에서 이 레포를 Import하면 자동 배포됩니다.

## 기능

- 대시보드 (기관별 현황, 마감 임박 공고, 인기 템플릿)
- 공고 탐색 (기관·분야 필터, AI 매칭 점수)
- AI 지원서 작성 (3단계 폼 → 스트리밍 생성)
- 합격 사례 DB (상세 + 다운로드)
- 지원서 피드백/점수 평가 (기관별 평가 기준)
